# `vola read`

Use this command when the user wants the contents or summary of one Vola path.

## Examples

- `vola read profile/preferences`
- `vola read project/demo`
- `vola read skill/writer/SKILL.md`
- `vola read secret/auth.github`

## Notes

- `project/<name>` is a summary view.
- A leading `/` is optional.
