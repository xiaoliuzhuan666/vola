# `vola ls`

Use this command when the user wants to browse the public Vola roots or a path under them.

## Notes

- `vola ls` lists the public roots: `profile/`, `memory/`, `project/`, `skill/`, `secret/`, `platform/`.
- A leading `/` is optional.
- Prefer `vola ls` over old `vola list` or `vola files ls` phrasing.
